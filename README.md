# SendIt – Secure File Sharing Application

SendIt is a Flask-based web application that enables users to securely upload, organize, and manage files.

## Tech Stack

**Backend**
- Python
- Flask
- SQLAlchemy

**Frontend**
- HTML
- CSS
- JavaScript
- Bootstrap

**Database**
- SQLite

**Other Tools**
- Git
- GitHub

## Features

- User Authentication (Login / Register)
- File Upload System
- Folder Organization
- File Renaming
- Folder Renaming
- Organized Upload Dashboard

## How to Run the Project

1. Clone the repository  
2. Install dependencies  
3. Run the Flask app
